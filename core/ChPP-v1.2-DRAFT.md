# 第一部分：核心公理层（A 系列）

## ChPP-A1 儿童是主体

儿童是主体，不因年龄、表达能力或决策能力而被降级为客体。

**主张能力（articulation capacity）≠ 主体资格（subjecthood）。**

主体资格是存在论事实，主张能力是发育变量。代理方的义务是适应主体的表达水平，而非以表达水平否定主体资格。

【与 A3 的接口】A3 代理四约束的前提是主体资格不可让渡；代理是功能代行，不是资格替代。

---

## ChPP-A2 决策权分层

决策权按不可代理性分为三层：

**不可代理层**：涉及身体完整性、基本安全、身份认同的核心决策，代理方无权代行，须以主体意愿为优先依据（即使主体为婴幼儿）。

**逐步参与层**：涉及日常生活安排、教育选择、社交关系的决策，代理方须根据主体年龄和认知水平逐步让渡决策权重，不得以"为你好"为由永久垄断。

**代理层**：涉及财产管理、法律程序、医疗紧急处置的决策，代理方可代行，但须事后向主体说明（主体具备理解能力时）。

---

## ChPP-A3 代理四约束

代理关系须同时满足四项约束：

**（1）保护≠替代**：代理方的保护行为不得替代主体的自主表达空间；

**（2）信息透明**：代理方须以主体可理解方式告知决策依据；

**（3）利益对齐审查**：代理方与主体存在结构性利益冲突时（如 A7 行为者），须引入独立审查；

**（4）动态退出**：代理关系须定期复核，主体表达退出意愿时须启动退出程序（A9 停止权接口）。

---

## ChPP-A4 自触发适配表达通道

DFN 内部程序须建立与主体发育水平适配的表达通道：

| 年龄段 | 表达通道 | 程序义务 |
|--------|----------|----------|
| 0–1 岁 | 身体信号（哭泣、肌肉紧张、回避眼神、身体后倾） | 程序方承担识别义务 |
| 1–3 岁 | 行为观察（指向、拒绝、依恋/回避模式） | 程序方记录并响应 |
| 3–6 岁 | 语言（简单因果、偏好表达） | 代理方须以可理解语言回应 |
| 6–12 岁 | 完整语言（复杂因果、假设性推理） | 程序须记录主体立场 |
| 12–18 岁 | 同等权（与成人同等的程序权利） | 独立代表资格（A10） |

【与 A8 的关系】垂直交叉——A4 描述表达能力发育顺序（"能说什么"），A8 规定质疑权分层开放（"有权要求什么"）。代理方不得以"表达不清"为由压制 A8-Q1。

---

## ChPP-A5 Cb 代理确认身体完整性升级

再生产劳动中的身体接触（Cb）代理须满足升级确认：任何涉及儿童身体的照护行为，在儿童具备表达能力后，须获得儿童的即时确认（非一次性授权）。

儿童可随时撤回确认，撤回须立即生效，代理方不得追问理由或施加压力。

---

## ChPP-A6 养育方式 vs 侵犯行为区分

以下情形不构成"养育方式"的合法范畴，须进入 ChPP 审查程序：

（1）以身体惩罚为常规管教手段；
（2）系统性地剥夺主体表达通道（如禁止哭泣、禁止表达不同意）；
（3）将主体作为情绪调节工具（如要求主体安抚代理方情绪）；
（4）以"隐私"为名阻止主体接触外部信息或支持系统。

---

## ChPP-A7 五种儿童特有 A 系叙事模板

代理方对儿童常用的五种 A 系叙事模板，审查方须识别：

| 模板 | 内容 | 识别信号 |
|------|------|----------|
| 被迫成熟 | "你比同龄人懂事"，要求儿童承担超出发育水平的情绪劳动 | 儿童回避同龄活动、过度关注成人情绪 |
| 感恩债务 | "我为你付出这么多"，将 Cb/Cc/Ce 转化为道德债务 | 儿童表达"欠父母"的自我归因 |
| 暴力即爱 | "打你是为你好"，将 A4 结构性掠夺重新编码为关怀 | 儿童将疼痛与亲密关联 |
| 比较降级 | "别人家的孩子…"，通过外部比较消解主体内在价值 | 儿童自我价值感依赖外部评价 |
| 隐私剥夺 | "你的一切都是我的"，以所有权逻辑否定身体主权 | 儿童无私人空间概念或过度防御 |

【L3-ChPP 扩展 3】审查方须识别代理方以下行为模式——信息截留、意愿替代、程序外施压（详见第三章）。

## ChPP-A8 质疑权（三层年龄递进）

**Q1（3 岁+，提问权）**：代理方须以主体可理解语言回应"为什么"，不得以权威压制。

**Q2（5 岁+，异议登记权）**：主体表达不同意须被记录，代理方决策须附简要理由。

**Q3（8 岁+，因果质疑权）**：主体可对决策理由提出因果性挑战，代理方须给出可检验的回应。上调至 8 岁因具体运算阶段稳定化通常完成于 7–8 岁。

**递进关系**：Q1→Q2 为自动触发（表达即触发记录），Q2→Q3 须经 L3 审查确认（非自动）。三层结构防止"质疑权"被简化为无约束的绝对否决，同时防止代理方以"年龄"为由压制 Q1。

**信心类型**：H（三层分解逻辑）；M（具体年龄数字，依赖发展心理学实证）。

---

## ChPP-A9 停止权

任何年龄主体在 DFN 内部任何程序（审查 / 裁决 / 培训 / 数据采集）中，有权以任何可识别方式（语言 / 行为 / 符号 / 身体信号）表达停止意愿，程序方须立即中止当前活动，不得要求主体提供理由或承受不利后果。

【与 MHP 0–3 接口】0–3 岁主体停止意愿通过身体信号（哭泣 / 肌肉紧张 / 回避眼神 / 身体后倾）识别，识别义务在程序方，不在主体。

---

## ChPP-A10 独立代表资格

12–18 岁主体在 DFN 内部任何涉及自身的程序（审查 / 裁决 / 集体决策）中，默认具有独立代表资格：

**（1）立场直接呈现**：主体立场直接呈现于程序，不经由代理方过滤或替代；
**（2）记录独立**：程序记录中主体立场独立于代理方立场；
**（3）分歧记录**：主体可持有与代理方不同立场，此分歧须被记录。

独立代表资格不依赖培训完成状态，不依赖代理方同意，不因代理方反对而失效。

【与 A3 代理四约束的接口（双层防御）】
**第一层（显性拒绝路径封堵）**：任何方不得以"你已有独立代表资格"为由拒绝提供程序支持。
**第二层（隐性降级路径封堵）**：独立代表资格的行使不构成任何方降低、撤回或延迟程序支持的正当理由。支持义务的履行以主体实际需求为准，不以代表资格的行使状态为转移。

**论据**：类别对称——A8（质疑权）与 A10 同属"内容层权利需独立公理化"类别，接受 A8 独立的理由同样适用于 A10。

## ChPP-A11 信息通道分离与数字代理权分层

### A11-1：5-MVS 最小可行方案（L3 审查分离）

**适用条件**：L3 审查对象与 6–12 岁主体的立场转述方为同一代理方。

**（1）立场采集分离**：6–12 岁主体立场采集须由独立于被审查代理方的人员执行，该人员不得是程序中任何利害关系方；

**（2）采集方式**：采用适合主体认知水平的方式（非书面陈述、非正式程序语言），具体方式由采集人员判断，须记录选择理由；

**（3）立场呈现**：采集结果以独立记录形式进入审查程序，标注"由独立人员采集，非代理方转述"；

**（4）局限声明**：MVS 解决信息通道分离，不构成 6–12 岁主体的独立代表资格。完整代表资格定义待 A10 稳定后展开；

**（5）独立人员最低资质**：非利害关系方。细化标准待 CALIBRATE-ChPP-3 培训者资格认证后对接。

### A11-2：数字环境代理权完整分层

**0-1岁：不可代理 → 全代理**
- **判定**：无符号表达能力
- **代理方**：法定监护人 + 儿保机构
- **数字门槛**：零数据采集，无平台账户，代理方仅可授权医疗级监测

**1-3岁：不可代理 → 监护人主导**
- **判定**：前语言/早期语言，无法形成可独立转述的立场
- **代理方**：监护人 + 早教专业人员
- **数字门槛**：采集仅限发展评估目的，禁止行为画像

**3-6岁：表达萌发 → 有限参与**
- **判定**：可表达偏好，不可评估后果
- **标准**：表达有效但不构成决策依据
- **代理方**：监护人决策，儿童表达作为必要输入（不可忽略）
- **数字门槛**：任何数字记录需监护人 + 独立第三方双重授权

**6-12岁：L3分离启动 → 逐步参与**
- **判定**：可形成立场，可初步评估后果
- **标准**：立场采集必须分离（5-MVS）
- **数字代理方资质**：儿童心理学认证 + 无平台利益关联 + 独立记录能力
- **关键规则**：儿童立场权重以表达有效性为判定标准，代理方不可归零
  - *来源：D0（决策权归主体）→ 权重趋近100%*
  - *来源：ChPP-A4（自触发适配）→ 权重以表达有效性为下限*

**12-18岁：接近完全代理**
- **判定**：可评估后果、可撤销同意
- **标准**：医疗/财务外，儿童为第一决策主体
- **代理方角色**：信息提供者，非替代决策者
- **数字门槛**：儿童自主管理数据权限，代理方可审计不可覆写

# 第二部分：数字环境操作化（ChPP-Digital）

## Task 1：数字 5-MVS 操作化

**TL;DR：** 平台同时是采集方和算法方，L3分离在平台内部物理不可能。平台外独立采集是操作化唯一解。

**推导链：**

```
L3审查要求：采集方 ≠ 被审计方
  → 儿童在平台表达立场时
  → 采集方 = 平台算法
  → 算法同步优化 = 对儿童建模
  → 采集方 IS 被审计方的优化器
  → L3分离在平台内部不成立
```

**操作化步骤（平台外实现）：**
1. **独立采集员**：非平台雇员、无算法接入权、儿童发展专业资质
2. **认知适配**：表达工具匹配发展阶段（非平台默认界面）
3. **独立记录**：采集记录写入平台外存储，标注采集者/方法/儿童年龄

**A4 阻断机制：**
- 平台算法将儿童表达归类为"娱乐内容"
- → 降权/限流/不进入公共讨论空间
- → 自触发适配表达通道被封堵在算法茧房内
- → 儿童表达无法触发真实世界的适配响应
- → A4失效：表达发生了，但系统不回应

**逻辑审计表：**

| 命题 | 逻辑结构 | 反驳谬误 | 信心 |
|------|----------|----------|------|
| L3分离平台内不成立 | 采集方=优化器→结构冲突 | "平台可以设独立部门"——组织隔离不等于数据隔离 | 0.92 |
| A4被算法封堵 | 表达→归类→降权→无响应 | 儿童可以绕过算法（直接联系）——但概率极低 | 0.85 |

**[CALIBRATE-ChPP-01]**：独立采集员资质门槛需参照ChPP-A11年龄分层细化。

## Task 2：A3 双层防御（数字环境）

**TL;DR：** 算法通过行为信号+语义分类隐性降级儿童表达。阻断需年龄感知算法+反降级审计。

**推导链：**

```
第一层（显性）：直接拒绝
  平台ToS限制儿童功能 → 可识别、可立法封堵

第二层（隐性）：算法降级
  机制1：行为信号降权
    快速滑动/低停留/无转化 → 算法判定为"低价值内容"
    → 儿童表达天然不具备成人的停留模式
    → 算法不区分"不成熟"和"发展阶段正常"
    → 儿童表达被系统性低估

  机制2：语义分类降格
    平台NLP将儿童语言模式标记为"低质量"
    → 降权→不进入推荐流→表达无公共影响力
    → ChPP-A10（独立代表资格）被算法静默取消
    → 不是拒绝，是沉默化
```

**阻断机制：**
1. **年龄感知算法**：检测到儿童表达模式时 → 禁止将其与成人表达做质量比较 → 走独立的"发展阶段适配"推荐通道
2. **反降级审计**：定期抽检被降权内容的年龄分布 → 若儿童内容降权率显著高于成人 → 算法歧视标记

## Task 3：TIFM R_i → R_c 级联（儿童特有θ路径）

**TL;DR：** 儿童认知架构未固化，R_i污染导致不可逆发育损害。θ_c窗口穿越在发育窗口期内发生。

**推导链：**

```
成人路径（可恢复）：
  成人已有完整信息架构
  → R_i 污染→认知负荷增加→但可用既有架构补偿
  → R_c 下降但未进入不可逆通道

儿童路径（不可恢复）：
  6-12岁 = 信息架构建构期（神经可塑性窗口）
  → 74.2%噪声环境中建构 = 用劣质材料建地基
  → 建构期的R_i损耗 ≠ 使用期的R_i损耗
  → 区别：成人丢了一本书，儿童从没建起图书馆
  → 47秒碎片化在发育窗口期→执行功能基础结构永久缩短

  θ穿越条件（儿童特有）：
    信息架构建构窗口期 ∩ R_i持续低于恢复阈值
    → 窗口关闭后架构固化
    → 即使R_i恢复，认知架构已定型为低完整性版本
    → 不可逆：不是"恢复慢"，是"窗口已关"
```

**逻辑审计表：**

| 命题 | 逻辑结构 | 反驳谬误 | 信心 |
|------|----------|----------|------|
| 建构期损耗≠使用期损耗 | 时间窗口×损耗→不可逆 | "神经可塑性持续终生"——6-12窗口强度远超成年期 | 0.88 |
| θ_c窗口穿越不可逆 | 窗口关闭+固化→不恢复 | 需要纵向发育追踪数据 | 0.75 |

**[SORRY-ChPP-01]**：θ_c的精确阈值依赖6-12岁信息架构发育的纵向追踪数据，当前TIFM为横向截面数据，无法独立验证窗口期不可逆假说。需与发育认知神经科学文献交叉验证。

**[SORRY-ChPP-02]**：6-12岁儿童注意力碎片化指数需要年龄分段数据，当前TIFM未提供年龄分段统计。F值不可直接用成人47秒数据填充。

## 四项联动结构

```
Task 1（5-MVS操作化）→ 解决"谁采集儿童立场"
Task 2（双层防御）   → 解决"算法如何静默取消Task 1的成果"
Task 3（R_i→R_c级联）→ 解决"如果不做以上两项，后果有多不可逆"

联合结论：
  两项制度设计（1+2）的时间窗口 = Task 3描述的θ_c窗口
```

# 第三部分：培训框架（ChPP-TR）

培训性质：**支持型**（降低信息不对称），非门槛型（不要求完成作为权利前提）。

| 要素 | 内容 | 类型 |
|------|------|------|
| TR-1 | DFN 内部程序流程（权利 / 程序 / 停止条件） | 支持型 |
| TR-2 | ChPP-A9 停止权的行使方式 | 支持型 |
| TR-3 | 自身立场如何被记录（程序透明） | 支持型 |
| TR-4 | 代理方可能的程序内行为模式识别（A7 模板 / 操控信号） | 支持型 |

**TR-4 归责缓解嵌入**："未能识别代理方的行为模式不是你的责任。识别操控是系统性保护的任务，不是个体的义务。即使接受了培训，识别失败的责任仍然在系统层面，不在你身上。"

**附则（信息隔离）**：培训完成状态不得在任何程序记录、裁决讨论或代表有效性评估中被引用、参考或暗示。裁决方不得在程序过程中知晓任何一方的培训参与情况。

# 第四部分：程序机制（L3-ChPP 扩展）

## L3-ChPP 扩展 2：自触发优先原则（IPP）

任何主体（包括 7 岁以上儿童）可直接启动 L3 审查程序并以自身立场呈现，无需代理方转述。此原则与 ChPP-A10 的功能等价延伸：自触发意味着程序启动资格与程序内立场独立性。

**注释**：L3 IPP 原始定义解决的是"谁可以启动审查"的问题；从启动资格到程序中立场独立性的推论是功能等价延伸，非 L3 IPP 原始陈述，此处显式标记。

---

## L3-ChPP 扩展 3：A7 压制三机制识别

审查方须识别代理方以下行为模式：

**信息截留**：阻止主体接触程序信息或独立支持方；
**意愿替代**：声称"我知道她真正想要什么"以覆盖主体表达；
**程序外施压**：在程序进行前后对主体施加压力以改变其立场。

任一机制被确认，代理方在该程序中的立场可信度降级，须引入独立验证。
