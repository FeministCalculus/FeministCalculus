第一章 最小公理系统（Minimal Axiom System）
═══════════════════════════════════════════════════════════════════════════════

CCST不建立在道德诉求或生物学决定论上，而是建立在关于"自维持系统"的物理约束与认知边界的最小公理集上。

───────────────────────────────────────────────────────────────────────────────
[AXIOM] CCST-P0（再生产循环不对称公理）
───────────────────────────────────────────────────────────────────────────────

任何能够长期维持自身存在的复杂系统（自维持系统），其内部存在再生产循环的结构性不对称。

  • 再生产循环节点：承担真实的再生产恢复成本，进入再生产循环（生理恢复、能量消耗、时间投入）。
  • 非再生产循环节点：不进入该循环，不承担对应恢复成本，但通过制度安排攫取循环成果。

成本的承担与成果的归属发生分离，是成本转嫁的结构性前提。CCST仅分析影响系统整体恢复速率的再生产不对称。

【结构性后果】P0在系统动力学层面的必然后果是形成恢复汇（Recovery Sink）：
          恢复流单向汇聚至特定结构，但不返回再生产循环。
          恢复汇不是外部节点，是系统内部参数偏置的涌现属性。

【术语注释】旧版"高/低恢复成本节点"已废弃。男性作为非再生产循环节点，
          不存在"恢复成本较低"的参与，而是不参与再生产循环。
          其"系统恢复者"身份是符号虚构。

───────────────────────────────────────────────────────────────────────────────
[AXIOM] CCST-D0（再生产决策主体公理）
───────────────────────────────────────────────────────────────────────────────

再生产决策权（是否启动、何时启动、在何种条件下启动再生产循环）在物理和物理-社会学层面不可剥夺地属于承担该再生产物质与能量成本的物理主体。

该公理纯粹描述物理成本的锚定，并不预设再生产行为必须发生或应该发生。

【防火墙】若主体决定再生产不发生 → 这是主体决策权的行使，不是系统失败。
          父权制的错误不是"阻止再生产"，是"剥夺再生产决策权"。

───────────────────────────────────────────────────────────────────────────────
[AXIOM] CCST-D1（提取与恢复速率递减公理）
───────────────────────────────────────────────────────────────────────────────

自维持系统维持自身运转需要消耗资源并具备恢复速率（Recovery Rate）。当非再生产循环节点对再生产循环节点的资源提取速率超过其自然的物理恢复速率时，该节点乃至系统整体的累积恢复速率将进入单调递减通道。

───────────────────────────────────────────────────────────────────────────────
[AXIOM] CCST-D2（不可逆物理阈值公理）
───────────────────────────────────────────────────────────────────────────────

系统的恢复速率存在一个不可逆的物理临界阈值 θ。一旦再生产循环节点的提取程度跨越该阈值（即超过 θ 点），系统恢复速率将断崖式坍塌，导致系统进入不可逆的退化或解体状态。

───────────────────────────────────────────────────────────────────────────────
[AXIOM] CCST-D3（认知不完全性公理）
───────────────────────────────────────────────────────────────────────────────

认知主体对任何自维持系统的整体状态、长周期成本以及 θ 阈值的具体物理边界，具有结构性的认知不完全性。换言之，系统内部的局部节点无法通过纯粹的即时理性计算来精准预知系统何时崩溃。

2.1 动态稳态与恢复汇形成
───────────────────────────────────────────────────────────────────────────────

  • 不创造恢复能力
  • 长期抽取恢复流
  • 不返回再生产循环
  • 内生（非外部入侵）

物理表现：
  再生产循环节点的恢复速率单调递减，进入衰竭。

───────────────────────────────────────────────────────────────────────────────
3.2 恢复抽取结构（Recovery Extraction Structure）
───────────────────────────────────────────────────────────────────────────────

恢复抽取结构是恢复汇在人类社会的制度实现。

定义条件：
  1. 位于系统内部（内生）
  2. 长期占有恢复收益
  3. 不承担等比例恢复贡献
  4. 扩张自身控制权

核心不是"恶意"，而是恢复流长期单向耗散。

与经典概念的区分：
  • 不是"寄生"（不需要外来/独立生命周期）
  • 不是"癌症"（不需要失控复制）
  • 不是"捕食"（不需要直接暴力）
  • 更接近制度经济学中的"抽取结构"（extractive institution）

  • 符号投资：男性投入符号资本（冠姓权、户籍登记、家族谱系写入、社会承认）。
  • 真实回报：要求的是真实回报（赡养、继承、服从、血统确认）。
  • 关键机制：符号ROI可以脱离真实再生产循环独立运行。

【与A6血统ROI的关系】

───────────────────────────────────────────────────────────────────────────────
3.6 Safe-to-Fail机制
───────────────────────────────────────────────────────────────────────────────

CCST不支持"零风险社会"，因为零风险 = 零探索，系统会失去 θ 认知能力。

因此需要 Safe-to-Fail Zone：
  • 允许局部失败
  • 必须阻止级联崩溃

必要条件：
  1. 结构隔离（失败不跨域传染）
  2. 时间隔离（短期失败不锁定长期路径）
  3. 可逆性评估（失败后可恢复）

MATH-1D 迟滞区（Hysteresis Zone）【审计后硬化】
───────────────────────────────────────────────────────────────────────────────

【声明】θ不是点，是不可逆区域。系统穿越拓扑分叉点后，即使退回原参数，也未必能回原态。

【符号修正】ℋ_k 是集合/区域，不是宽度。ℋ_k 的宽度记为 w_k = θ_k,upper - θ_k,lower。

定义开关状态变量 σ_k ∈ {0, 1}：

    σ_k(t) = 
    ⎧ 0,  if R_k(t) ≥ θ_k,upper          （可恢复态S_1，级联耦合C_jk不激活）
    ⎨ 1,  if R_k(t) ≤ θ_k,lower          （不可恢复态S_2，级联耦合C_jk激活）
    ⎩ σ_k(t-Δt), if R_k(t) ∈ (θ_k,lower, θ_k,upper)  （迟滞区ℋ_k，保持历史状态）

其中：
  θ_k,lower：正向拓扑分叉点（S_1 → S_2）
  θ_k,upper：逆向拓扑分叉点（S_2 → S_1），通常 θ_k,upper > θ_k,lower
  ℋ_k = { R_k | θ_k,lower < R_k < θ_k,upper }：迟滞区（集合）
  w_k = θ_k,upper - θ_k,lower：迟滞区宽度（标量）

路径依赖（Memory Effect）：
  当R_k(t)处于ℋ_k内时，系统状态不由当前R_k决定，而由历史路径决定：
    • 若系统从S_1进入ℋ_k（σ_k=0→保持0直到触及θ_k,lower）
    • 若系统从S_2进入ℋ_k（σ_k=1→保持1直到触及θ_k,upper）
  这正是Preisach算子的核心特征。

能量势垒（Energy Barrier）：

  定义 E_{1→2}：系统从可恢复态S_1穿越至不可恢复态S_2所需能量。
  定义 E_{2→1}：系统从不可恢复态S_2返回可恢复态S_1所需能量。

  迟滞区存在的物理条件：

    E_{2→1} >> E_{1→2}

  即：逆向穿越需要远大于正向穿越的外部能量输入。

  修复失效条件（D2的数学表达）：

    ∂R_k/∂I_k → 0  意味着常规投入I_k无法提供E_{2→1}级别的能量。
    必须有大剂量外部能量注入（社会重构、代际周期、生态工程）才能穿越ℋ_k。

迟滞区与级联传播的关系：
  若R_j进入ℋ_j且σ_j=1，则级联耦合C_jk保持激活，
  即使R_j短暂回升至θ_j,lower以上（仍在ℋ_j内，σ_j不翻转）。
  系统需要获得E_{2→1}级别的外部能量输入，使R_j ≥ θ_j,upper，才能关闭C_jk（σ_j翻转为0）。

8.1 反例1：高恢复率但低自由度社会
───────────────────────────────────────────────────────────────────────────────

【案例设定】
某社会：R_b高（TFR>2.5）、R_e稳定、R_n高（制度协调强），
        但R_c极低（认知压制严重）、R_i极低（信息高度同步化）。

【CCST回应】
  1. 不将"低自由度"自动判为系统失败。
  2. 分析：R_b高是否依赖R_c和R_i的过度抽取？
     • 若R_b的维持需要持续压制R_c/R_i → 存在隐藏恢复汇（恢复抽取结构通过认知压制抽取R_c/R_i以维持R_b）
     • 若R_b的维持不依赖R_c/R_i抽取 → 该配置在CCST框架内是稳态的
  3. CCST不预设"自由"是恢复的必要条件。
     "自由"属于不可映射域（价值判断，非恢复变量）。
  4. 若该社会长期维持动态稳态且无恢复汇 → CCST在此案例上部分失效
     （CCST无法解释其稳定性，需其他理论补充）。

【结论】CCST允许"高恢复率+低自由度"作为可能的稳态配置，
      但要求验证各分量之间的耦合关系。
      若确实无恢复汇，则该案例构成CCST的边界案例。

───────────────────────────────────────────────────────────────────────────────
8.2 反例2：低生育但高技术自动化社会
───────────────────────────────────────────────────────────────────────────────

【案例设定】
AI照护替代人力、R_b低（TFR<<1）、R_e高（生态效率高）、
R_c高（认知恢复强）、自动化维持R_n。

【CCST回应】
  1. 关键问题：R_b低是否由恢复汇导致？
     • 若是（决策权被剥夺、照护成本私有化）→ θ_b被突破，系统进入结构重组
     • 若否（主体自主选择不生育、AI社会化照护充分）→ R_b低是D0决策权的行使，不是系统失败
  2. 关键问题：AI照护是否创造新的恢复汇？
     • AI照护若被资本控制、抽取数据价值 → 恢复汇从A4迁移至A3（数字平台配置）
     • AI照护若社会化、无抽取 → 不构成恢复汇
  3. D0防火墙：再生产决策权属于主体，不预设"必须发生"。
  4. 关键问题：低生育是否进入θ_b迟滞区？
     • 若TFR已进入ℋ_b → 即使未来政策逆转，恢复至更替水平需要跨代周期
     • 若TFR未进入ℋ_b → 存在恢复窗口

【结论】低生育本身不触发θ_b。触发θ_b的是"恢复汇对R_b的抽取导致局部修复失效"。
      若主体自主选择+AI社会化照护无抽取，则该配置在CCST框架内是稳态的。
      这正是D0防火墙的设计目的。

8.3 反例3：强宗教社会（高同步化、低异见、长期稳定）【证伪条件硬化】
───────────────────────────────────────────────────────────────────────────────

【案例设定】
某社会：S(t)→1（高同步化）、Diversity<<d_min（低异见）、
        但已维持500年稳定，R_e/R_b/R_n均维持动态稳态。

【CCST回应】
  1. MATH-7的同步指数S(t)是预警指标，不是结构重组判定。
  2. 高同步化+长期稳定的可能解释：
     • 该社会的恢复机制拓扑与同步化兼容（如小规模、低耦合、强本地冗余）
     • 或：该社会存在CCST未建模的恢复机制（理论边界）
  3. 不可映射域接口：宗教体验属于MATH-1C的双重不可约域+结构性突变，
     CCST不分析宗教本身，只分析宗教对R(t)的结构性突变效应（若可观测）。

【去道德化声明】CCST不将"高同步化"自动判为"恶"。
              若该社会确实无恢复汇且长期稳态，则它在CCST框架内是合法状态。

【证伪条件3.1——不可逃避的经验证伪标准】
  若发现一个现代大型复杂社会（高维、高耦合、强依赖跨域协作），满足：
    (a) S(t) → 1 且 Diversity < d_min（认知完全同步且无异见）
    (b) R_i 和 R_c 长期稳定不退化（信息完整性和认知恢复能力维持稳态）
    (c) 系统未表现出向外部转嫁再生产成本的行为（无恢复汇证据）
    (d) 系统已维持该状态超过两个代际周期（排除短期波动）

  则CCST的P0公理（再生产循环不对称导致恢复汇）及整个数学推导链
  **彻底宣告破产**。

  【Lakatos保护带爆破】此证伪条件不可被以下辅助假设豁免：
    × "该社会属于不可映射域"（不可映射域不豁免P0的物理约束）
    × "该社会规模小、低耦合"（条件已限定"现代大型复杂社会"）
    × "CCST允许自身失效"（失效必须有具体条件，此条件已给出）
    × "存在隐藏恢复汇"（条件(c)要求无恢复汇证据）

【结论】CCST在此案例上不是"部分失效"或"需要扩展"，而是"硬核破产"。
      这是理论真诚的终极测试。

8.4 失效条件与边界（综合）
───────────────────────────────────────────────────────────────────────────────

CCST可能失效于：
  1. 恢复能力无限（已否定）
  2. 不存在阈值（已否定）
  3. 完全认知可达（已否定）
  4. 代理不产生权力（已否定）
  5. 存在不依赖恢复汇的现代大型复杂社会长期稳态（证伪条件3.1）
  6. 所有文化现象均可被量纲化为恢复变量（PROP-11证伪条件）
  7. 系统结构重组均可通过撤回原参数完全恢复（PROP-12证伪条件）
  8. 所有不可映射域现象均可被参数化为恢复系统的可加扰动项（PROP-15证伪条件）
  9. 所有复杂系统的状态均可由当前R_k完全决定（PROP-16证伪条件）

目前没有经验支持条件5-9。

CCST分析 vs CCST证明：
  CCST分析：说明某现象符合CCST结构。
  CCST证明：证明CCST是唯一解释。
  后者目前不存在。因此CCST是竞争性框架，不是封闭意识形态。

失效条件：
  CCST可能失效于：恢复能力无限 / 不存在阈值 / 完全认知可达 / 代理不产生权力。
  目前没有经验支持这些条件。
