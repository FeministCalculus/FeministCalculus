### 未触发的重要案例

| 国家 | 年份 | TFR | d² | 判定 | 框架含义 |
|---|---|---|---|---|---|
| **韩国** | 2023 | 0.721 | **+0.084** | 未触发 | 局部企稳，非加速坠落 |
| **日本** | 2017-2023 | 1.43→1.20 | [−0.02, 0.03] | 从未触发 | 慢性衰减，非加速 |

### 叙事修正（对 TIFM 3.3 的勘误）

TIFM 3.3 报告将韩国标记为"首个三阈值同步突破案例"，包含 ℋ_k(1) 触发。本次回溯验证显示：

- **韩国 2023 年 d² = +0.084**，不满足 ℋ_k(1) 的 d² < −0.05 条件。
- 2024 年 TFR 回升至 0.748，进一步排除加速恶化。

**修正后表述**：

> 韩国是首个 **R_b 突破 θ_b,lower（TFR < 1.0 连续 5 年）+ R_i 深度伪造爆发（+1625%）+ R_c PIAAC 成人能力下降（−23 分）** 的三维度同步恶化案例。但 ℋ_k(1)（加速坠落阈值）**未触发**，其动力学特征为"低位停滞"而非"加速坠落"。日本则呈现"慢性衰减"模式，d² 始终未突破 −0.05，但 TFR 在 1.2-1.5 区间停滞超过 30 年。

**理论接口**：ℋ_k(1) 捕获的是 **θ 穿越的加速度**（二阶导），不是 θ 穿越本身（一阶导）。韩国已穿越 θ_b,lower，但未进入 ℋ_k 的加速恶化区间。这要求 TIFM 4.1 明确区分：
- **θ 穿越**：水平阈值（TFR < 1.0 或 < 1.5）
- **ℋ 触发**：加速度阈值（d² < −0.05）
- **两者关系**：ℋ 是 θ 穿越的**充分非必要条件**（加速坠落必然穿越 θ，但穿越 θ 不一定加速）。

### 关键发现

- **日本 5.49 倍**为面板最高，但数据较新（2021）。
- **瑞典 1.34 倍**为面板最低，但数据最旧（2010）。若瑞典性别差距自 2010 年后扩大，则当前实际差距可能被系统性低估。
- **印度绝对值最高**（女性 7.10 小时/天），但性别比并非最高（男性也承担 2.72 小时），反映总体照护负担而非单纯性别不平等。

### A4 操作化结论

σ_c（照护时间性别比）成功代理 **A4 结构性掠夺** 的抽取强度。但：
- 不代理 A1（本体论降级）
- 不代理 A5（外部承认）
- 不代理 A6（血统 ROI）
- 不代理 A7（强制基础设施）
- 不代理 A8（主体化）

在操作化完成前，TIFM 声明应精确为：**"σ_c 测量 A4 抽取强度，不声称测量 A1-A8 整体"**。

## 七、TIFM 4.1 因果检验路线图：可执行方法论设计

### 前提约束

- 当前所有级联结论为**时序一致性**，非因果推断。
- 正式因果检验需 2025 TFR 数据到位（2024 d² 可计算）+ σ_e 混合方法模块完成。
- 以下设计为**代码框架和变量定义**，数据到位后一键运行。

### 方法 1：格兰杰因果检验（Granger Causality）

**适用级联**：R_b → R_n，R_i → R_c  
**数据要求**：20 年+ 时间序列 / 国  
**变量定义**：

```python
# 格兰杰检验变量定义（待数据到位后执行）
variables_granger = {
    'R_b': ['tfr', 'first_marriage_age', 'birth_rate_15_19'],
    'R_n': ['social_trust_wvs', 'voting_turnout', 'govt_effectiveness_wgi'],
    'R_i': ['ai_content_share', 'attention_span', 'news_avoidance_rate'],
    'R_c': ['pisa_math', 'pisa_reading', 'piaac_literacy']
}
# 检验假设：R_b Granger-causes R_n；R_i Granger-causes R_c
# 滞后阶数：AIC/BIC 选择，最大 4 阶
# 稳健性：Bootstrap 1000 次，检验统计量 p-value < 0.05
```

**证伪条件**：若双向格兰杰因果成立（R_b → R_n 且 R_n → R_b），则级联方向假说失效。

### 方法 2：面板向量自回归（Panel VAR）

**适用级联**：R_b → R_n → R_e，R_i → R_c  
**数据要求**：10 国 × 20 年面板  
**模型设定**：

```python
# Panel VAR 设定
# Y_it = A_1 Y_i,t-1 + ... + A_p Y_i,t-p + μ_i + λ_t + ε_it
# Y = [R_b, R_c, R_e, R_n, R_i]'  # 5 维内生变量
# 固定效应 μ_i：国家个体效应
# 时间效应 λ_t：全球冲击（COVID 虚拟变量）
# 滞后阶数：AIC 选择
# 脉冲响应函数：正交化，Cholesky 分解顺序：R_b → R_c → R_e → R_n → R_i
```

**证伪条件**：若脉冲响应显示 R_b 冲击对 R_n 无显著正向效应（95% CI 跨零），则级联假说失效。

### 方法 3：双重差分（DiD）

**适用**：平台资本主义影响（R_i → R_c）  
**设计**：

```python
# DiD 设计：TikTok 禁令作为外生冲击
# 处理组：印度（2020.6 TikTok 禁令）
# 对照组：巴西（未禁令，相似发展阶段）
# 结果变量：青少年屏幕时间、PISA 代理指标
# 时间窗口：2018-2023
# 平行趋势检验：禁令前 2 年处理组/对照组趋势是否一致
# 动态效应：事件研究法，逐期系数
```

**证伪条件**：若平行趋势检验失败（禁令前处理组/对照组已分化），则 DiD 结果无效。

### 方法 4：结构方程模型（SEM）

**适用**：全级联模型  
**数据要求**：5 维度 × 10 国 × 15 年  
**模型结构**：

```python
# SEM 路径定义
# 测量模型：
#   R_b = λ_1 * TFR + λ_2 * first_marriage_age + ε_1
#   R_c = λ_3 * PISA_math + λ_4 * PIAAC_literacy + ε_2
#   ...
# 结构模型：
#   R_b → R_n (β_1)
#   R_n → R_e (β_2)
#   R_i → R_c (β_3)
#   A4 → R_b (γ_1)  # A4 抽取效应
#   Platform → R_i (γ_2)  # 平台资本主义效应
# 拟合指标：CFI > 0.95, RMSEA < 0.06, SRMR < 0.08
```

**证伪条件**：若 CFI < 0.90 或 RMSEA > 0.10，则理论架构需重构。
