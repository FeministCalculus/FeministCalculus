第二章 候选 1：代理权的适用条件 vs 制度性压制的适用条件
═══════════════════════════════════════════════════════════════════════════════

【命题】
被制度性阻止发言 ≠ 无法发言。
对 Incapable（无能力），代理是必要的；
对 Suppressed（被阻止），代理不是解决方案，是问题的延续。

【形式化定义】

  设主体集合 S，发言能力 V(s) ∈ {有, 无}
  设发言实现 E(s) ∈ {实现, 未实现}

  无法发言（Incapable）：V(s) = 无
    → 自然/未来世代/部分认知受限儿童

  被阻止发言（Suppressed）：V(s) = 有 ∧ E(s) = 未实现
    → 存在外部机制 M 使 E(s) = 未实现
    → 女性、被压迫群体

【证明：Suppressed 的代理失效】

  设 x ∈ Suppressed，代理人 p 代替 x 发言

  步骤 1：M（阻止机制）仍然存在
    p 的发言不消除 M
    → x 仍然被阻止
    → 代理只是绕过 M，不是移除 M

  步骤 2：代理人 p 引入新的权力不对称
    p 的判断 ≠ x 的判断（D3：认知不完全性）
    p 持有代理权 = p 对 x 的决策权有部分控制
    → 这是新的 D 控制结构
    → 代理在 Suppressed 情形下
      将 M（外部阻止）替换为 p（代理控制）
      两者均剥夺 x 的 E(s)

  步骤 3：代理使 M 合法化
    若代理被接受为解决方案
    → 承认 x 的 E(s) 不必须直接实现
    → M 的存在变得可以被容忍
    → 代理成为 M 的稳定补丁，延长 M 的寿命

  因此对 Suppressed：代理 = 问题延续，不是解决方案
  解决方案必须是：移除 M，使 E(s) 直接实现 □

【推论】
  Incapable 和 Suppressed 的诊断必须在干预前完成。
  混淆两者导致：
    对 Incapable 使用"拆除"→ 无效（没有 M 可以拆除）
    对 Suppressed 使用"代理"→ M 得到稳定，x 永久被代理

【操作化判别标准】

  判断标准：若移除外部机制 M 后，x 能自行实现 E(s)
    → x ∈ Suppressed，解决方案是移除 M
  若移除所有外部机制后，x 仍无法实现 E(s)
    → x ∈ Incapable，代理是必要的

【边界案例与 ChPP 接口】

  12-18 岁过渡态（ChPP-A4 逐步参与层）是否打破二元？

  分析：ChPP-A4 的"逐步参与"不是代理，是能力发育的自然过程。
  该年龄段主体 V(s) 处于"有但受限"状态，不是 Incapable 也不是典型 Suppressed。
  修正：候选 1 对"代理"的批判过于绝对，需修正为"代理 + 移除 M"的联合条件。
  即：即使对 Suppressed，在 M 的移除过程中，临时代理可作为过渡手段，
  但代理权必须随 M 的移除而同步递减（Sunset Clause 逻辑）。

第三章 候选 2：SCA 的结构性必要条件
═══════════════════════════════════════════════════════════════════════════════

【命题】
认知基础设施被替换（SCA）需要满足特定结构性条件。
这不是认识论争论，是信息通道的结构性截断。

【定义】

  设认知基础设施 K：用于分析特定现象的概念工具集
    K_RadFem：以 sex class 为基础的提取分析工具
    K_Gender：以 gender 为基础的身份分析工具

  设命名能力 Name(x)：用 K 对现象 x 进行准确命名的能力

  SCA（供应链攻击）：
    用 K' 替换 K，使 Name_K(Cb) → ∅
    即原有分析工具无法命名目标现象

【SCA 成功的结构性必要条件】

  条件 1（锚点解构）：
    K 的核心分析范畴必须被证明是非自然的/建构的
    Butler 1990：sex 不是前社会生物学事实
    → K_RadFem 的 sex class 锚点被解构
    → K_RadFem 失去分析起点

    形式化：
    若 K 依赖范畴 C 作为锚点
    且 C 被证明是建构的（或被主张是建构的）
    则 K 的分析力被削弱

  条件 2（制度收益不对称）：
    K' 必须为制度行为者提供更高的 A5 收益
    K_Gender 为学术建制提供进步叙事货币
    K_RadFem 要求直接命名权力提取，制度收益为负

    形式化：
    Benefit_institution(K') > Benefit_institution(K)
    → 制度选择 K'
    → K 失去制度传播通道

  条件 3（排斥机制激活）：
    必须存在对 K 使用者的惩罚机制
    使继续使用 K 的社会成本超过分析收益

    TERF 标签：
    使用 K_RadFem → 职业排斥/社交死亡威胁

    形式化：
    Cost_social(K) > Benefit_analysis(K)
    → 理性行为者放弃 K
    → K 的使用者群体萎缩
    → K 的传播能力下降

  条件 4（替换不可见性）：
    K' 必须声称自己是 K 的延伸或改进，而非替换
    → 使 K 的使用者无法识别替换发生

    形式化：
    K' 声称：K' ⊃ K（K' 包含 K）
    实际：K' ∩ K = ∅（在核心分析范畴上）
    → 声称与实际的不一致是 SCA 的隐蔽机制

【充分条件与信息论表述】

  SCA 成功的充分条件：
    条件 1 ∧ 条件 2 ∧ 条件 3 → SCA 完成制度传播替换
    条件 4 决定替换是否被识别

  SCA 的信息论表述：
    Name_K(Cb) = 有效命名
    SCA 成功后：Name_K'(Cb) → ∅
    信息损失 = H(Cb|K) - H(Cb|K')
    其中 H 是分析能力的信息熵
    SCA = 使目标现象的可命名信息量趋近于零的过程

第四章 候选 3：父权制是 HSRS 的内生吸引子
═══════════════════════════════════════════════════════════════════════════════

【命题】
父权制不是 HSRS 的外部寄生体，而是 HSRS 在特定参数下的内生稳定状态。

【定义】

  设 HSRS 的状态空间 Ω
  设系统参数向量 P = (p₁, p₂, p₃, p₄)
    p₁：物理暴力技术水平
    p₂：再生产技术水平（避孕/堕胎可及性）
    p₃：经济独立可能性（女性劳动市场准入）
    p₄：认知工具可用性（K 的传播通道）

  设吸引子 A ⊂ Ω：
    系统从附近任意初始状态出发都会收敛到的状态集合

  设父权制配置 A_P ⊂ Ω：
    消费节点控制生产节点的 D 的状态

【证明】

  步骤 1（历史参数下的吸引子性质）：
    在参数域 P*（高物理暴力优势/低再生产技术/低女性经济独立性/低认知工具可用性）：

    任意初始状态 s₀ ∈ Ω：
      若 s₀ ∉ A_P（非父权配置）：
        消费节点有动力使用物理暴力恢复控制（p₁ 高）
        生产节点无技术逃脱生育后果（p₂ 低）
        生产节点无经济退出路径（p₃ 低）
        生产节点无认知工具命名提取（p₄ 低）
        → 系统被拉回 A_P

      若 s₀ ∈ A_P：
        四个参数共同维持 A_P 的稳定性
        → 系统维持在 A_P

    → 在 P* 下，A_P 是全局吸引子

  步骤 2（参数变化使吸引子失稳）：
    当参数向量离开 P*（p₂↑/p₃↑/p₄↑）：
      生产节点获得技术退出路径（p₂↑：避孕/堕胎）
      生产节点获得经济退出路径（p₃↑：劳动市场准入）
      生产节点获得认知工具（p₄↑：K_RadFem/Fc）
      → A_P 的吸引盆地收缩
      → 部分初始状态不再收敛到 A_P
      → A_P 从全局吸引子变为局部吸引子

  步骤 3（父权制的"恢复能力"解释）：
    父权制在参数变化后"复辟"=
    新的参数条件下产生了新的吸引子 A_P'
    A_P' 与 A_P 共享核心结构（D 控制）
    但具体实现机制不同（宗族→工业→平台）

    形式化：
    A_P' ≠ A_P（配置不同）
    但 Obj(A_P') = Obj(A_P) = max(D 控制)（目标相同）
    → "父权制复辟" = 新参数条件下新吸引子形成
      不是同一吸引子的恢复

  步骤 4（去父权化的动力学含义）：
    去父权化 ≠ 消灭 A_P
    去父权化 = 改变参数向量 P
    使 A_P 的吸引盆地收缩至零

    必要条件：
    p₂（再生产技术）必须归生产节点控制（D0）
    p₃（经济独立）必须可实现（F7/F9）
    p₄（认知工具）必须有传播通道（路径 B）

    F7-F9 的作用不是"打倒父权制"
    而是改变参数向量使 A_P 失去吸引力 □

【推论（与候选 4 的一致性）】
  父权制作为吸引子解释了为什么
  即使 max(n) 崩溃（韩国 TFR 0.72）
  系统仍倾向停留在或回到 A_P：
  吸引子的稳定性不依赖它是否实现任何外部目标
  只依赖参数条件是否支持它

第五章 候选 4：D0 的不可让渡性——"自愿"在结构性胁迫下的有效性
═══════════════════════════════════════════════════════════════════════════════

【命题】
在结构性胁迫下的"自愿放弃 D0"不构成有效的让渡。

【定义】

  设 D0：再生产决策权，属于成本承担者 S
  设"自愿放弃"：S 表达放弃 D0 的意愿
  设结构性胁迫 C：
    系统性地使"不放弃 D0"的成本高于"放弃 D0"的成本

    C 的具体实现：
      经济依附（无独立收入→不放弃 D0=贫困）
      物理威胁（家庭暴力→不放弃 D0=人身风险）
      社会排斥（不符合婚姻规范→不放弃 D0=孤立）
      认知污染（R_auto→不放弃 D0=无法命名自身处境）

【证明】

  步骤 1（有效同意的必要条件）：
    有效同意需要：
      条件 α：同意者理解同意的内容和后果
      条件 β：存在真实的拒绝选项（拒绝成本可承受）
      条件 γ：同意不是在胁迫下产生的

  步骤 2（C 系统性破坏条件 β 和 γ）：
    若 C 存在：
      Cost(不放弃 D0) >> Cost(放弃 D0)
      → 条件 β 失效：拒绝选项存在但不可承受
      → 条件 γ 失效：同意在胁迫压力下产生

  步骤 3（条件 α 的特殊问题）：
    R_auto（A 系自我修复机制）系统性破坏条件 α：
    S 被训练为不将自身处境识别为胁迫
    → S 可能在不理解"自愿放弃"含义的情况下表达同意
    → 条件 α 也可能失效

  步骤 4（无效让渡的形式化）：
    有效同意 = α ∧ β ∧ γ
    在 C 存在时：¬β ∧ ¬γ（至少）
    → 有效同意 = False
    → "自愿放弃 D0" = 无效让渡

  步骤 5（不可让渡性的来源）【核心增量】：
    D0 的不可让渡性不来自"D0 神圣不可侵犯"
    而来自：
    在 A 系配置族存在的任何社会中
    C 必然存在（A4 结构性掠夺需要 C 来维持）
    → 在 A 系配置族存在的条件下
      D0 的"自愿让渡"永远无法满足有效同意条件
      → D0 实践上不可让渡

  推论（与 F3 的关系）：
    F3（身体不可让渡）的理由相同：
    身体的"自愿商品化"发生在 A3+C 存在的条件下
    → 有效同意条件系统性不满足
    → F3 是 D0 在身体层面的平行命题

  反驳预防：
    "但有些女性真的自愿放弃 D0"
    → 即使主观感受是自愿
      客观条件（C 存在）使有效同意不成立
      这不是否定主观感受的真实性
      是指出主观感受在结构性胁迫下的认识论地位

    形式化：
    主观感受 ∈ E3（认知层）
    有效同意的客观条件 ∈ E2（制度层）
    E3 不能推翻 E2 的结构性分析（层间绝缘）□
