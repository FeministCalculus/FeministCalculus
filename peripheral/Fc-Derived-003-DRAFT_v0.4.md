第二章 命题 1：A1-A5 内生张力定理
═══════════════════════════════════════════════════════════════════════════════

【命题】
A5 承认需求随 A1 执行强度上升而发生形态滑移（连续谱模型），
形成 A 系配置的内生张力回路。

【A5 连续谱模型（v0.3 新增）】
A5 需求从"需要真实主体承认"到"不需要主体只需权力展示"是连续变化，
A1 执行强度是驱动变量，不需要精确切分 A5-α/β 两个模态。
可观察中间状态：
  A5-α 端：寻求正面反馈，关注回应质量
  中间态：追求回复数量/愤怒程度而非内容质量（南拳博主行为变化的可测量节点）
  A5-β 端：行为本身即满足，不依赖受害者任何回应
连续谱操作化：测量 A1 强度与 A5 满足条件之间的相关性，
不需要切分两个模态（CALIBRATE-D003-1 难度降低）。

【推导链】

```
前提 1（A5 定义）：
  A5 外部承认 = 同性社会声誉货币
  A5 的有效性依赖：承认来源被判定为有意义的主体

前提 2（A1 定义）：
  A1 本体论降级 = 主体性被制度性悬置
  A1 执行 → 承认来源的主体性下降

推导：
  A1 执行强度 ↑
  → 承认来源（女性）主体性 ↓
  → A5 承认货币面值 ↓（来自非主体的承认价值递减）
  → A5 承认需求未被满足
  → 寻求更多承认
  → 更激烈执行 A1（降级更多，以获得更多控制感）
  → 承认来源主体性进一步 ↓
  → 循环

  A1↑ → A5-α（主体承认型）需求受挫
  → 切换至 A5-β（权力展示型）
  → 回路以变形形态维持（非崩溃）
  → A5-β 满足度递减时趋向边界
  → 边界处（小花梅）在 Fc 射程外，推导链仅描述趋势

【边界标记】：
  P1 终点"A5 完全失效"在 Fc 边界外（附录 H）。
  框架内只能描述"趋向边界时 A5-α→A5-β 的形态滑移"，
  不能验证终点。
```

**两种 reliable 的区分**（mimo 审计追加）：

| 类型 | 机制 | 结构 | 特征性失败模式 |
|------|------|------|--------------|
| A4 结构下的"可靠" | 被提取者 A8 内化，无意识顺从，不知道自己被降级 | A4 产出，不证明 P4 | 结构变化时突然崩溃（顺从无内生适应力） |

**可区分性注释**（mimo 二轮审计追加）：

A4 顺从无内生适应力，结构变化（如 A4 闭环被扰动）时突然崩溃；
F5 协作包含拒绝权，结构变化时通过主动协商维持稳定。
这是 P4 的可检验预测，不只是定性区分。

P4 的适用条件：被降级者对降级有认知反应时才成立。
完全 A8 内化者仍可能"可靠"（无意识顺从），但这是 A4 的产出，不是 F5 的实现。

### 结构对抗

**测试**：A1-A5 内生张力是否依赖 Fc 框架内部定义，无法独立验证？

**结果**：部分是。P1 依赖 A5 的"主体性依赖"假设——若 A5 的满足不需要真实主体
（如纯粹的权力展示），则回路不成立。这是真实的边界条件，不是分析缺失。
[CALIBRATE-D003-1]：需要操作化区分 A5 的两种形态：
  A5-α（主体承认型）：需要来源具有主体性
  A5-β（权力展示型）：不需要来源具有主体性
  若 A5-β 是主导形态，则 P1 失效。

### 极端对抗

**测试**：若 A1 完全执行（小花梅边界），A5 是否真的失效？

**结果**：这是框架边界测试。小花梅已在 Fc 外部（附录 H）。
但在趋向边界的过程中（A1 降级程度增加），
A5 的形态是否从"主体承认型"转变为"权力展示型"，
是一个需要独立观察的问题。
[CALIBRATE-D003-2]：A1 降级程度与 A5 形态转变的关系待操作化。

### 反向对抗

**测试**："奴隶生不出人"是女权命题，是否本身就是规范性的，
不能用于描述性推导？

**结果**：命题 3 的推导是描述性的——
它说的是 A1 配置下代际 A8 传递会腐蚀 A6 的意义基础，
不是说"这是错的"。规范层评价（F1/F2）单独成立，不进入推导链。
这是层间绝缘的应用。

### 历史失败样本

**测试**：历史上是否有 A1 配置下 A6 投资意义完整保留的案例？

**结果**：
- **古罗马奴隶制**：奴隶主的后代继承奴隶主身份，但奴隶后代
  在法律上不被视为延续奴隶主血统——A6 的血统意义在 A1 极端执行下
  确实被制度性切断。
- **中国封建宗族制**：妾生子在宗族地位上低于嫡子，母亲 A1 降级
  直接传导到后代的宗族地位——A6 ROI 因 A1 配置而内部分层。
- **当代拐卖案例**（用户观察）：被拐卖妇女的子女对母亲的降级内化，
  是 A8 代际传递的直接观察，来源权重 B（用户观察+媒体报道）。
