## 1. 命题1：再生产决策权是第一序独立变量

**定义**：
- 设再生产行为 R ∈ {发生, 不发生}
- 设再生产决策权 D ∈ {主体持有, 外部持有}
- 设成本承担者 S（有子宫的物理主体）

**命题**：
D 是第一序独立变量，其归属不由 R 的发生状态、也不由系统短期恢复速率直接决定。

**证明**：

假设 D = f(R)：
- 即"是否发生再生产"决定"谁有决策权"
- 这是倒果为因：R 的发生本身需要 D 先被确定
- 若 D 依赖 R，则 D 的确定需要 R 已知
- 而 R 的确定需要 D 已知
- → 循环依赖，f 不存在

假设 D = g(系统恢复速率)：
- 即"系统需要恢复"决定"谁有决策权"
- 这是 A6 的逻辑（系统需要后代，所以控制生育）
- 但 P0（G2' 操作化延伸）说成本承担者物理上承受恢复成本
- 成本锚定与系统状态无关
- → g 的存在违反 P0

因此 D 不是任何其他变量的函数。
D 是第一序独立变量。□

**注释 §1.1**：P0 与 G2' 的关系为"操作化延伸"——G2' 描述物种层面再生产劳动的必要性，P0 将该必要性锚定到个体成本承担者的物理主体上。非严格同构，层级距离保留。

## 2. 命题2：系统边界的政治性

**定义**：
- 设物理边界 B_p：由 E1 物理过程决定（流域、大气、生态群落）
- 设社会边界 B_s：由 E2 制度安排决定（国家、市场、社区）
- 设影响范围 I(x)：干预行为 x 的实际物理影响范围

**命题**：
B_s 的设定是权力分配的结果，且在 I(x) > B_s 时，边界设定等价于成本承担权的单方分配。

**证明**：

步骤1（B_s 可以不等于 B_p）：
- 存在反例：国家在 B_s 内进行 E1 实验，污染通过大气/水流扩散至 B_s 外
- I(x) > B_s
- → B_s ≠ B_p，反例存在

步骤2（B_s 的设定涉及权力分配）：
- 若 I(x) > B_s：B_s 内的主体获得实验收益，B_s 外的主体承担实验成本
- → 成本-收益跨边界分离（P0 的跨边界版本）
- 谁设定 B_s 决定了谁承担成本
- 设定 B_s = 分配成本承担权
- 分配成本承担权 = 权力行使
- → B_s 的设定是权力分配的结果

**规范跳跃**（D0/F3 桥梁，非纯实证）：
- D0：成本承担者持有决策权
- B_s 设定决定成本承担者是谁
- → B_s 设定权必须包含被影响方
- 单方设定 B_s 等价于单方决定成本由谁承担
- → 违反 D0（成本承担者应持有决策权）
- → 因此 B_s 的设定需要被影响方参与

□

## 3. 命题3：父权制的核心目标是控制再生产决策权，而非最大化再生产数量

**定义**：
设目标函数 Obj_A（父权制主导目标函数）

**命题**：
在现代条件下（高机会成本 + 低死亡率 + 避孕技术普及），Obj_A = max(对 D 的控制程度)，而非 max(n)。

**证明（反证法证伪假设1）**：

如果 Obj_A = max(n)，则：

**预测1**：当技术使 n 可以在 D 归主体的条件下最大化时，父权制应支持该技术。
- 反例：现代辅助生殖技术（精子库、试管婴儿）可以在无男性控制的条件下实现高 n
- 但父权制制度持续限制单身女性/同性伴侣使用该技术
- → 预测1 不成立

**预测2**：当主体自愿选择高生育率时，父权制应支持而非干预。
- 反例：女性自愿生育但要求经济独立和决策权
- 父权制制度持续设置障碍（婚姻捆绑、堕胎限制）
- → 预测2 不成立

**预测3**：堕胎限制的目标是增加 n。
- 反例（大熊猫类比的形式化）：大熊猫可自行终止妊娠 → 雄性控制无效 → n 不可控
- 人类女性有堕胎技术 → 若目标是 max(n)，则应支持堕胎（可终止不良妊娠，优化质量）
- 但父权制系统性限制堕胎
- → 限制堕胎不是优化 n，是剥夺 D
- → 预测3 不成立

假设1（Obj_A = max(n)）被三个反例证伪。

**证明假设2（Obj_A = max(对 D 的控制程度)）**：

**观察1**：所有被证伪的预测都有一个共同结构——父权制在"保持对 D 的控制"和"增加 n"之间选择前者。

**观察2**：A6（血统ROI）的核心不是 n 本身，是"确定性"——知道 n 是自己的后代。
- 确定性需要控制 D（控制女性的性和生育选择）
- → A6 = max(确定性) = max(对 D 的控制)

**观察3**：A5（主体性依赖）——"我需要有人在我之下才能确认我在之上"。
- 需要主体在下方 → 需要主体不持有 D
- → A5 = max(对 D 的控制)

**假两难封闭**：
- 若 max(D) 是 max(n) 的手段，则当手段与目的冲突时，手段应被放弃
- 测试：精子库技术使 n 在 D 归主体条件下可最大化
- 预测：若 max(n) 是目的，则控制 D 应被放弃（采用精子库）
- 观察：父权制系统性限制单身女性使用精子库
- → 放弃手段的条件满足，但手段未被放弃
- → max(D) 不是手段，是独立目标

**更强的测试**：
- 若 max(D) 是手段，则在 max(n) 已实现的条件下，父权制应放松 D 控制
- 观察：即使生育率极低（韩国 TFR 0.72），父权制仍收紧 D 控制（堕胎限制、生育惩罚）
- → n 已崩溃，但 D 控制加强
- → 手段在目的消失后仍存在
- → 手段已转化为独立目标
- → 假两难排除

因此：
- A6 + A5 → Obj_A = max(对 D 的控制程度)
- 且 max(对 D 的控制) 与 max(n) 在现代条件下冲突
- 父权制选择前者
- → Obj_A = max(对 D 的控制程度) □

**注释 §3.2 历史适用域限定**

命题3 的诊断效力依赖"现代条件"——高机会成本、低死亡率、避孕技术普及、辅助生殖技术可及。在此条件下，max(D) 与 max(n) 可被分离测试，父权制系统性选择前者。

前现代条件下，两者存在混叠：高死亡率使 n 对 D 控制有工具依赖，D 控制通过制度垄断（婚姻/继承/家父权）而非技术禁止实现。max(D) 与 max(n) 的冲突不尖锐，命题3 的分离测试不可行，但 A 系诊断（D 控制存在）仍有效。

此限定非削弱，是适用域精确化：提升可证伪性，明确边界。公理层（F 系）不受此限定——F3 身体主权是规范公理，非历史诊断。
