链 1：活体悖论（A3 → A1）✅ HARDENED v9.4.1

骨架：
A3 货币完备性：再生产劳动可估算价值，可货币化
+ 具身性约束：再生产劳动与承担者身体物理不可分
→ 货币化必须将"产生该劳动的身体能力"纳入交换对象
→ 等价于将活人身体纳入购买方生产函数
→ 主体性被制度性悬置（A1 激活，维度①：身体可被合法支配）
→ 自由缔约人格被摧毁
→ A3 应用于具身性劳动时自我拆解

结论：¬F3（身体可让渡）→ A3 扩张无阻断 → A1 激活 → ¬F1
等价命题：F1（主体平等）→ F3（身体不可让渡），F3 是 F1 的结构性必要条件

边界：具身性约束是 G' 层物理前提（非规范）。若技术完全解除具身性（全人工子宫 + 自动化育儿），此前提失效，F3 需重新定义。

链 2：D0 不可让渡性 ✅ HARDENED

骨架：
D0：决策权锚定于承担物理成本的物理主体
→ "放弃决策权" = 委托他人决定不可代偿的物理成本
→ 有效委托需满足 αβγ：
    α = 理解（知道自己在放弃什么）
    β = 真实选项（有可以不放弃的现实选项）
    γ = 非胁迫（不在威胁下决定）
→ 在 A 系配置族中：
    C（结构性胁迫）系统性破坏 βγ
    R_auto（内化机制）破坏 α
→ αβγ 在 A 系配置族中无法同时满足
→ "自愿放弃 D0" 在 A 系配置族中无效
→ D0 实践上不可让渡

结论：A 系配置下不存在有效的"自愿放弃再生产决策权"。

边界："A 系配置族"边界待精确化。若存在 A 系弱配置（如北欧），链可能局部不成立。

链 5：Obj_A = max(D) ✅ HARDENED（Fc-Derived-001）

骨架：
若 Obj_A = max(n)（最大化后代数量），则预测：
  预测1：支持辅助生殖技术（提高 n）→ 实际：系统性限制女性自主使用 ✗
  预测2：支持堕胎（终止不良妊娠，优化 n 质量）→ 实际：限制堕胎 ✗
  预测3：A6 寂灭后放弃控制（n 已崩溃，手段应退出）→ 实际：n 崩溃后控制加强 ✗

三个反例同时成立
→ Obj_A ≠ max(n)

正向建构：
A6 = max（血统确定性）= max（对 D 的控制）
A5 = 需要主体在下方 = 需要主体不持有 D
A6 + A5 → Obj_A = max（对 D 的控制程度）

结论：父权制的核心目标是控制再生产决策权本身，不是生育数量。这解释了为什么 TFR = 0.72 的韩国仍在加强对女性身体的控制。

证伪条件：发现父权制在 TFR 崩溃后主动放松控制且支持女性自主辅助生殖。
